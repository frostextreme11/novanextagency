Ini adalah alamat website kamu (contoh: `namabisnismu.com`).

* **Tips:** Pilih nama yang singkat, mudah dieja, dan merepresentasikan brand kamu. Hindari penggunaan angka atau tanda hubung kalau bisa.
* **Ekstensi:** `.com` masih jadi raja, tapi `.id` (Indonesia) makin populer dan dipercaya untuk pasar lokal.

### 2. Hosting (Tanah/Lahan)

Kalau domain adalah alamat, hosting adalah tanah tempat kamu membangun toko. Di sinilah semua file foto produk, teks, dan data website kamu disimpan.

* **Penting:** Jangan tergiur hosting gratisan untuk bisnis! Kecepatan (loading speed) dan keamanan (security) adalah segalanya di 2026. Pembeli nggak akan nunggu website yang loadingnya lebih dari 3 detik.

### 3. Platform CMS (Kerangka Bangunan)

Kita akan menggunakan **WordPress**. Kenapa? Karena 43% website di seluruh dunia pakai WordPress. Gratis, mudah digunakan, dan punya ribuan *plugin* pendukung.

## Tutorial Langkah demi Langkah: Cara Membuat Website Toko Online ğŸ’»

Sudah siap domain dan hosting? Yuk, ikuti langkah-langkah praktis ini.

### Langkah 1: Install WordPress

Di panel hosting kamu (biasanya cPanel), cari fitur "Softaculous" atau "WordPress Installer". Klik install, isi data login (username & password), dan *voila*! Website dasar kamu sudah jadi dalam hitungan menit.

### Langkah 2: Install Plugin WooCommerce

WordPress itu ibarat rumah kosong. Biar jadi toko, kita butuh etalase dan kasir. Di WordPress, alatnya bernama **WooCommerce**.

* Masuk ke Dashboard WordPress > Menu **Plugins** > **Add New**.
* Cari "WooCommerce".
* Klik **Install Now** lalu **Activate**.
* Ikuti *setup wizard* untuk mengatur mata uang (Rupiah), alamat toko, dan jenis produk.

### Langkah 3: Pilih Tema (Desain Toko)

Jangan biarkan tampilan toko kamu kaku. Pilih tema yang responsif (bagus di HP) dan cepat.

* Rekomendasi tema gratis: **Astra**, **OceanWP**, atau **Kadence**.
* Caranya: Menu **Appearance** > **Themes** > **Add New**.

### Langkah 4: Upload Produk

Ini bagian serunya! Masukkan foto produk, deskripsi, harga, dan stok.

* **Tips Foto:** Gunakan foto resolusi tinggi tapi *file size* kecil (dikompres). Foto gelap dan buram = tidak ada penjualan.
* **Tips Deskripsi:** Jangan cuma tulis spesifikasi. Tulis manfaat! Ganti "Bahan Katun Combed 30s" dengan "Bahan adem, menyerap keringat, nyaman dipakai seharian meski cuaca panas."

### Langkah 5: Setup Pembayaran & Pengiriman

Untuk pemula, kamu bisa setting pembayaran via Transfer Bank Manual (BCA, Mandiri, dll). Tapi biar kelihatan profesional, integrasikan dengan *Payment Gateway* (seperti Midtrans atau Xendit) agar pembeli bisa bayar pakai QRIS, E-Wallet, atau Kartu Kredit dan konfirmasi otomatis.

Untuk ongkir, install plugin seperti **WooCommerce Ongkir** (misal dari plugin Ongkos Kirim Indonesia) yang terhubung dengan API RajaOngkir, jadi hitungan ongkir JNE/J&T/SiCepat bisa otomatis keluar.

Belajar dari kesalahan orang lain itu lebih murah daripada belajar dari kesalahan sendiri. Hindari ini:
